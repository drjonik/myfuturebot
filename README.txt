Инструкция по запуску:

1. Установи зависимости:
   pip install -r requirements.txt

2. Укажи токен Telegram бота как переменную окружения:
   В Windows:
     set BOT_TOKEN=твой_токен

   В Linux/macOS:
     export BOT_TOKEN=твой_токен

3. Запусти бота:
   python bot.py

Для Render.com:
- Подключи GitHub-репозиторий с этим проектом
- Укажи переменную окружения BOT_TOKEN
- Render сам подхватит .render.yaml и запустит бота
